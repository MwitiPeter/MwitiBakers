# Mwiti Bakers

Production-ready commerce platform for Mwiti Bakers.

## Phase 5 setup

- `apps/web`: Next.js storefront and dashboard shell
- `apps/api`: Express API and service layer
- `packages/shared`: shared types and constants

## Getting started

```bash
npm install
npm run dev
```

## Scripts

- `npm run dev:web`
- `npm run dev:api`
- `npm run build`
- `npm run typecheck`
- `npm run smoke`
